# 💕 随机浪漫表白网页

一个超级浪漫的表白网页项目，每次刷新都会呈现不同的视觉效果和表白文案，让你的告白充满惊喜与感动。

## 📖 项目简介

这是一个纯前端的表白网页项目，专为想要用特别方式向心爱的人表达心意的你而设计。通过随机组合的背景、文案、动画效果，这个网页拥有超过 **10,000种** 不同的展示组合，确保每一次打开都是独一无二的浪漫体验。

无论是在电脑、平板还是手机上打开，都能获得完美的视觉体验。只需要一个浏览器，就能将你的心意传递给TA。

## ✨ 为什么选择这个项目

与其他表白网页不同，这个项目的独特之处在于：

- **每次刷新都不重复** - 通过10种渐变背景、10种浪漫标题、10条深情文案和4种动画效果的随机组合，创造出无限可能
- **完全响应式设计** - 无论在什么设备上打开，都能完美适配屏幕，保持最佳观看效果
- **零门槛使用** - 无需安装任何依赖，无需服务器，直接用浏览器打开就能使用
- **精心设计的动画** - 包含漂浮的心形、闪烁的星星、粒子特效等多种浪漫元素，让表白更加生动

## 🎯 主要功能特性

- **随机背景渐变** - 10种精心挑选的浪漫配色方案，从紫色梦幻到粉色温馨，每一种都独具魅力
- **多样化文案库** - 包含10条真挚动人的表白文案，从含蓄温柔到热烈直白，总有一句能打动TA的心
- **丰富的动画效果** - 心形飘落、星星闪烁、粒子漂浮，或是三者混合，营造梦幻浪漫的氛围
- **交互式烟花特效** - 当TA点击"我愿意"按钮时，页面会绽放出绚丽的烟花，庆祝这个美好时刻
- **完美的移动端体验** - 使用弹性布局和响应式字体，在手机上表白同样精彩

## 🚀 快速开始

### 使用方法

1. 下载项目文件到本地
2. 找到 `index.html` 文件
3. 双击文件，或者右键选择用浏览器打开
4. 就是这么简单！网页会自动加载并展示随机的浪漫效果。
